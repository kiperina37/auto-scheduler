# 智能日程管理器 (Auto-Scheduler) 🚀

这是一个基于 **React + TypeScript** 开发的现代化 Web 应用。它不仅能记录你今天要做的事，还能根据任务时长和优先级，为你规划出一整天的时间表。

## ✨ 核心功能

- **智能排序**：自动根据任务优先级（高/中/低）进行排序，确保最重要的事先做。
- **自动排时**：只需输入时长和开始时间，应用会自动计算每个任务的结束时间。
- **极简 UI**：支持现代浏览器的自适应布局，视觉清新。

## 🛠️ 技术栈

- **前端框架**: React 18
- **开发语言**: TypeScript
- **构建工具**: Vite
- **样式设计**: Vanilla CSS

## 🏃 如何运行？

如果你想在本地机器上运行此项目，请确保已安装 [Node.js](https://nodejs.org/)。

1. **克隆仓库**
   ```bash
   git clone https://github.com/zhenglijia686-tech/auto-scheduler.git
   cd auto-scheduler
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动项目**
   ```bash
   npm run dev
   ```

4. **查看效果**
   打开浏览器访问 [http://localhost:5173/](http://localhost:5173/)。
